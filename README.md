# MotionBall

Tiny Python motion project built with Pygame.

A bunch of colorful bouncing balls moving around the screen at terminal velocity like absolute crackheads.

---

## Features

- Multiple animated balls
- Random colors
- Collision with screen boundaries!
- Smooth 60 FPS motion
- Built fully through terminal workflow

---

## Tech Stack

- Python 3
- Pygame
- Git
- GitHub
- Questionable engineering decisions

---

## How To Run

Clone the repository:

```bash
git clone https://github.com/CaptainK1s/MotionBall.git
cd MotionBall
```

Create virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
python3 -m pip install pygame
```

Run app:

```bash
python3 app.py
```
